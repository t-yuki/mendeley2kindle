Usage:
1. To execute Menedeley2Kidnle, you should double click mendeley2kindle.jar (Windows)
 or type java command on your console
  java -jar mendeley2kindle.jar --help

2. Open your Mendeley database in local computer:
for example, please click Open Menedeley DB button and select the file:
C:\Users\%USER%\AppData\Local\Mendeley Ltd\Mendeley Desktop\%USERMAILADDRESS%@example.com@www.mendeley.com.sqlite (Windows Vista or later)

3. Connect your kindle to computer and Select your Kindle device to export Mendeley collections

4. Click "Export to Kindle" button
Please note that THIS SOFTWARE IS CURRENTLY EXPERIMENTAL-PREALPHA VERSION AND QUALITY. SO CONTENTS OF YOUR KINDLE MAY BE ACCIDENTIALLY DESTROYED BY UNKNOWN REASONS OR BUGS.
